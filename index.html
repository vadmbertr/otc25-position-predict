Objective

The goal of this mini-competition is to predict the position of 5 drifters one week into the future using their last observed week trajectory and collocated physical fields. The drifters are deployed during the OTC25 cruise in the North Atlantic Ocean.

Rather than predicting the exact position of the drifters, you are asked to estimate for each drifter an area of research where it should be found.
Several rounds of the competition will be held every week for the same set of drifters. The participant with the most successes across rounds and drifters will be declared the winner.

Rules

  • Each participant can submit only one prediction file per round (participants can update their prediction during a round: if several predictions are attempted for the same round, the last attempt is used).
  • A round starts at 8.00 AM and ends at 6.00 PM on the same day.
  • A prediction consists of a GeoJSON file containing one "feature" per drifrer.
  • Each "feature" must have a "geometry" field with a "Polygon" type.
  • Each "feature" must have a "properties" field with a "text" property whose value is the id of the drifter.
  • Each "Polygon" must have an area smaller or equal to 100 km^2
  • The prediction file must be uploaded using this form.
  • The prediction file must be uploaded before the 6.00 PM deadline.

Getting started

For each round (8.00 AM to 6.00 PM):
  1. Visit the Ocean Virtual Laboratory portal.
    1. Play with the displayed data and the time wheel.
    2. Draw your zones of research.
    3. Download them as a GeoJSON file.
  2. Submit your predictions before 6.00 PM using this form.
  3. At the end of the round, check the updated leaderboard.
